# LinkedIn Content System Delivery (v2)

This archive contains the content system export for Notion import.

Folders:
- Post Drafts: 100_Posts.md
- Bonus Assets: supporting markdown files

How to import into Notion:
1. Create a new page in Notion.
2. Drag & drop markdown files into the page.
3. Use Notion board/table to organize posts into a Kanban.
