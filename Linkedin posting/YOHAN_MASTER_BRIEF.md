## My Core Beliefs (Content Foundation)

- Most brands install Gorgias but never truly monetize it
- **Statistics are massively underused in e-commerce CX** — this is a content differentiator
- **Chat on the website is underestimated** — both for support and pre-sale conversion
- **TPO (Tickets Per Order)** is the most underrated CX metric — tracks CX quality vs volume
- Automation should start with low-risk, high-volume tickets — not complex flows
- CX is one of the most undervalued and under-leveraged functions in e-commerce
- Brands say they're "customer-first" — their support setup tells a very different story
- Hiring more agents is almost never the right answer — fix the system first
