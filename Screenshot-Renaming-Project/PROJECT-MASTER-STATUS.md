## Proven Process (From Jan & Feb Work)

### Step 1: Locate Files
- Find directory containing month's PNG files
- Count total files
- Verify file naming (typically generic or simple)

### Step 2: Batch Analysis (5-10 files per batch)
- Open each file individually
- Describe actual content: platform, people, companies, topics
- Propose detailed name in format: `[PLATFORM]-[PERSON/COMPANY]-[SPECIFIC-CONTEXT]-[TIMESTAMP].png`

### Step 3: Present to User
- Show batch analysis with visual descriptions
- Display proposed detailed names
- Get verification/approval

### Step 4: Apply Renames
- Only rename after user approves batch
- Track which files have been renamed
- Document any patterns found

### Step 5: Move to Next Batch
- Repeat until all files for month are complete
- Move to next month

## File Naming Pattern (Proven Format)

All renamed files follow this structure:

```
[PLATFORM]-[PERSON/COMPANY]-[SPECIFIC-CONTEXT]-[TIMESTAMP].png
```

### Components:
- **PLATFORM:** Slack, Email, HubSpot, Gorgias, Website, LinkedIn, Document, etc.
- **PERSON/COMPANY:** Names visible in screenshot (e.g., "Deel", "Giulia-Scotucci", "Michaela-Weber")
- **SPECIFIC-CONTEXT:** What the content is about (e.g., "HR-vacation-days", "partner-links", "meeting-reschedule")
- **TIMESTAMP:** File's original timestamp (YYYY-MM-DD-HH.MM.SS)

### Real Examples:
- ✅ `email-Deel-HR-vacation-days-balance-confirmation-2025-02-18-16.00.39.png`
- ✅ `slack-Julien-Marcialis-insights-IM-team-discussion-2025-02-06-16.33.03.png`
- ✅ `document-Direct-Intro-Referral-Form-partner-lead-submission-agency-2025-02-25-10.45.30.png`
- ✅ `website-LinkedIn-profile-Bermet-Yamaliova-Markut-Head-of-Growth-bold-2025-02-03-11.06.19.png`
- ✅ `HubSpot-contact-Laurence-Cobby-deals-activities-2025-02-17-11.48.37.png`

### Anti-Patterns (What NOT to Do):
- ❌ Generic names: `email-message.png`, `chat-conversation.png`
- ❌ Made-up details: Names/companies that don't appear in the screenshot
- ❌ Vague context: `document-content.png`, `file-interface.png`
- ❌ Simplified versions: Using `email-HR-review` when actual content is `email-Deel-HR-vacation-days-balance`

## Known Issues & Solutions

### Issue: Mount Disconnections
- **Cause:** Reading too many files at once (10+ files)

### Issue: Simplified Names in Summary
- **Cause:** Not using detailed proposed names
