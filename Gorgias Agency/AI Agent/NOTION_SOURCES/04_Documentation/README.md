# AI Agent Setup & Optimization Playbooks

**Last Updated:** February 24, 2026
**Purpose:** Comprehensive Notion-based playbooks for Claude AI to reference when working with customers on AI Agent audits, recommendations, and optimizations.

---

## 📌 What This Is

These documents are designed for **Claude AI to reference and use** when:
- **Conducting AI Agent audits** for customers (analyzing performance, identifying gaps)
- **Making optimization recommendations** (providing specific improvement strategies)
- **Setting up AI Agent implementations** (guiding merchants through configuration)
- **Refining AI Agent performance** (improving automation rates, CSAT, handover handling)
- **Analyzing customer support workflows** (understanding current state vs. potential)

**Claude's Role:** Use these playbooks as authoritative reference materials to provide structured, proven guidance based on Notion documentation that has been validated through real customer implementations.

**Last Notion Update:** February 10, 2026

**Purpose:** The primary 3-step implementation process with detailed call scripts, time allocations, and merchant guidance.

- Structuring implementation calls with specific timing and talking points
- Following a proven methodology for customer success

**Key Sections:**
- 🎯 Context & Goal (10% uplift in automation rate)
- 🚀 Methodology (Vitally notes, Calendly links, templates, macros)
- Step 1: Kick-off (45' call with detailed prep, call script, follow-up)
- Step 2: Follow-up (60' call with refinement, performance review)
- Step 3: Wrap-up (30' call with finalization and knowledge transfer)
- 📊 Auditing Accounts section

**Key Information:**
- Detailed call scripts with exact time allocations
- Discovery questions for each step
- Shopping Assistant configuration by merchant AOV
- Actions, Flows, Order Management setup guidance
- Objection handling tactics
- Performance metrics to track
