#### Best Practices for Successful Character Replacement
**Character Image Preparation:**
- Always use high-quality avatar images (1024x1024 or higher)
- - Ensure consistent lighting between character image and reference video background
- - Match clothing and styling to the motion reference content (if doing product showcase
with specific outfit)
- - Maintain similar character positioning in image (not too close/zoomed)
- - Use professional quality avatar generation for best results
**Reference Video Selection:**
- Choose source videos with clear, distinct movements
- - Avoid overly complex or rapid motion for first attempts
- - Test orientation: vertical works best for TikTok/Reels, horizontal for broader
compatibility
- - Ensure reference video has good lighting and clear subject
- - Prefer videos where motion is centered in frame
**Motion Transfer Quality:**
- Simpler, slower movements transfer better than rapid/complex choreography
- - 5-15 second clips usually produce best results
- - Longer videos (15-30s) work but may accumulate slight errors
- - Character size in image should approximately match reference video subject size
**Common Mistakes to Avoid:**
- Using blurry or low-quality character images
- - Mismatched aspect ratios between image and video
- - Reference videos with multiple subjects (Motion Control works best with single
character)
- - Overly complex dance or rapid motion in first attempts
- - Ignoring lighting consistency between components
- - Not checking orientation requirements for platform (TikTok vs YouTube vs Instagram)

**Issue: “Character doesn’t match the motion”**
- Solution 1: Check aspect ratio - resize image to match video dimensions
- - Solution 2: Adjust character position in image - should be centered and occupy similar
frame space as reference video subject
- - Solution 3: Try shorter reference video (under 10 seconds) for simpler motion patterns
**Issue: “Motion transfer creates unnatural movements”**
- Solution: Use simpler, clearer motion in reference video

- - Avoid rapid head movements or extreme gestures
- - Test with slower choreography first
**Issue: “Avatar looks inconsistent frame-to-frame”**
- Solution 1: Use higher resolution character image (1024px+)
- - Solution 2: Ensure consistent lighting in character image matching video background
- - Solution 3: Check that Motion Control model is selected (not basic video generation)
**Issue: “Processing is very slow”**
- Solution 1: Reduce video length (try 5-10 seconds instead of 30)
- - Solution 2: Lower resolution setting if available

### The Optimal Prompt Formula for Each Task Type
#### IMAGE PROMPTS: The Avatar Formula
**Structure:**
[Identity + Age] + [Visual Style] + [Clothing] + [Expression] + [Context] + [Technical Specs]
**Example Template:**
“[Professional woman, 28 years old], [modern, sophisticated], [wearing tailored blazer in navy
blue], [confident, approachable smile], [clean studio background with soft gradient], [4K
professional photography, sharp focus, natural lighting, highly detailed face, commercial
photography quality]”
**Why This Order Works:**
1. Identity first - establishes WHO
2. 2. Style second - establishes aesthetic direction
3. 3. Clothing - brand/context specific
4. 4. Expression - personality/tone
5. 5. Context - environment/setting
6. 6. Technical specs - quality parameters
**Critical: Use Specific Descriptors**
❌ Wrong: “attractive woman”
✓ Right: “woman with sharp jawline, clear skin, natural makeup, intelligent expression”
❌ Wrong: “nice background”
✓ Right: “soft-focus white background with subtle shadow, studio lighting setup”
#### VIDEO PROMPTS: The Scene Formula
**Structure:**
[Subject] + [Action/Movement] + [Setting] + [Visual Style] + [Duration/Pacing] + [Camera Work] +
[Technical Specs]
**Example Template:**

“[Woman in professional attire], [walking while reviewing mobile device, then looking up with
realization], [modern office environment with floor-to-ceiling windows and afternoon sunlight],
[cinematic, professional, modern aesthetic], [10 seconds total, paced to match audio], [steady
camera following slight motion, shallow depth of field], [1080p, 24fps, natural color grading,
subtle professional tone]”
**Pro Tips:**
- Always specify duration (Genspark varies 2-30 seconds)
- - Describe movement clearly (“walks across frame”, “turns to camera”)
- - Mention pacing relative to audio if syncing
- - Include camera technique (static, dolly, pan)
- - Specify emotional tone (energetic, calm, urgent)
#### CHARACTER REPLACEMENT PROMPTS: The Motion Transfer Formula
When using Kling V2.6 Motion Control, your prompt should focus on character consistency, not
motion (motion comes from reference video):
**Structure:**
[Character Description] + [Clothing/Styling] + [Lighting Match] + [Emotional Context]
**Example:**
“Professional woman avatar, consistent facial features from provided image, wearing matching
outfit (black blazer, professional styling), matching studio lighting with soft shadows, confident
and professional demeanor throughout motion sequence”
**Key Difference:**
- Don’t describe the ACTION (video provides it)
- - DO ensure character consistency specifications
- - DO match lighting of reference video

### Common Prompt Mistakes (And How to Fix Them)
#### Mistake 1: Too Vague
❌ “Create a video of someone using a product”
✓ “Create a 12-second TikTok video: Woman in her 20s, professional casual outfit, holding
wireless earbuds, inserting them in ear, happy surprised expression, modern apartment
background, morning sunlight, energetic, high quality, 9:16 vertical, optimized for TikTok”
**Why:** Vague prompts = random results. Specific prompts = consistent results.
#### Mistake 2: Conflicting Styles

❌ “Photorealistic but with cartoon elements and watercolor effects”
✓ “Photorealistic with subtle artistic brushstroke overlay” (or pick ONE style clearly)
**Why:** Models struggle with conflicting artistic directions.
#### Mistake 3: Impossible Physics
❌ “Person levitating while walking naturally”
✓ “Person walking on tightrope with perfect balance, focused expression”
**Why:** Be clear about what’s physically happening.
#### Mistake 4: Over-Complicated Scenes
❌ “Woman with 5 people in background, multiple products, complex lighting, 3 different
locations”
✓ “Woman in foreground (sharp focus) with blurred office background, holding one product,
professional lighting”
**Why:** More isn’t better. Clearer simpler scenes = better quality.
#### Mistake 5: Missing Technical Specs
❌ “Make a cool video”
✓ “Make a cool 15-second video for TikTok (9:16 vertical), fast-paced, energetic, 720p minimum
resolution, modern aesthetic”
**Why:** Technical specs prevent format issues and poor quality.

### Iterative Refinement Strategy
**First Generation Review:**
Look at output and note:
- Did model understand the core concept? ✓/✗
- - Quality level acceptable? (1-10 scale)
- - Any major issues? (lighting, proportions, style)
- - Partially correct but needs adjustment?
**Refinement Approach:**
| Issue | Refinement Strategy |
|---|---|

| Wrong person/subject | Describe person MORE specifically (facial features, age, body type) |
| Wrong style/aesthetic | Add style reference (“like magazine photography” or “editorial style”) |
| Wrong emotion/tone | Specify expression more clearly, add emotional context |
| Quality too low | Specify quality markers (“high detail”, “professional grade”, “4K sharp”) |
| Wrong background | Describe background specifically (“modern office with windows”, not “nice
background”) |
| Proportions wrong | Use comparative references (“realistic proportions”, “anatomically correct”)
|
| Inconsistent with brand | Add “matches brand aesthetic for [brand description]” |
**Iteration Limit:**
- Avatar/image: 2-3 iterations max (then try different model)
- - Video: 1-2 iterations (then try different prompt approach)
- - Stop if: Fundamental model limitation (use different model instead)
