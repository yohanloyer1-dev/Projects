## Client Overview

**Company:** Accessory Partners LLC
**Industry:** Automotive aftermarket (trucks, off-road, performance parts)
**Business model:** Multi-brand, B2C + B2B (wholesale dealers)

**Main brands:** Suspension Superstore (core), MaxTrac, All American Billet, Legacy Products (sunroofs), Vision X (lighting)

**Tech stack:** BigCommerce (multi-store), Gorgias, ShipStation, ERP (NOT connected — major gap)

---

## Key Stakeholder

**Rob** — Primary operator, handles most support himself. Time-constrained. WhatsApp preferred. Decision-maker.

---

## Core Problems

- 5–6 brands in ONE Gorgias account, single inbox (`help@`), no routing/separation
- ERP disconnected → fragmented data, manual lookups
- Tracking broken: multiple packages/order, no unified view
- SMS: phone numbers missing +1 → Gorgias can't match customers → Rob reassigns manually (can't fix at source)
- Gorgias massively underutilized (no Rules, Macros, Flows, CSAT)
- Rob = only agent, no bandwidth to improve system

**Retainer ✅:** ~$600–900/month

## Implementation Roadmap

1. **WISMO System** (Priority #1 — highest ROI)
   - Auto-detect intent, pull tracking from ShipStation
   - Handle: no tracking yet / in transit / delivered / multi-package

2. **Multi-brand Structure**
   - Separate email per brand, auto-tagging, views, routing logic

3. **Returns Flow**
   - Structured chat/form intake, pre-qualification

4. **Chat Flows**
   - B2C vs B2B, returns, tracking, FAQs

5. **CSAT + Reporting**
   - Activate CSAT, track touches/ticket, response time, satisfaction

6. **ShipStation ↔ Gorgias integration**

7. **SMS workaround** — normalize +1, reduce manual reassignment
