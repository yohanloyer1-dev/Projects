# Gorgias Platform Glossary

**Last updated:** 2026-03-21

---

## Core Concepts

| Term | Definition |
|------|-----------|
| **AI Agent** | Gorgias's autonomous AI that handles customer tickets end-to-end without human involvement |
| **Automation Rate** | % of all tickets fully resolved by AI with no human touch |
| **Coverage Rate** | % of tickets the AI Agent attempts to respond to |
| **Success Rate** | % of tickets the AI Agent resolves without human involvement |
| **Handover** | When AI Agent transfers a ticket to a human agent |
| **Handover Topic** | Configured intent that forces AI to always transfer (e.g. "legal threat") |
| **Ignore Rule** | Rule that prevents AI from attempting to respond to a ticket |
| **Guidance** | AI Agent knowledge instruction — tells AI how to handle a specific intent |
| **Snippet** | Short pre-written response template (manual use) |
| **Macro** | Response template + action bundle (can auto-close, tag, assign) |
| **Rule** | Automation trigger: if [condition] then [action] |
| **Flow** | Guided conversation path in Gorgias chat widget |
| **View** | Filtered ticket inbox (e.g. "All open tickets for Brand X") |
| **Tag** | Label applied to tickets for routing/reporting |
| **CSAT** | Customer Satisfaction score — sent after ticket resolution |
| **TPO** | Tickets Per Order — ratio of support volume to order volume. Most underrated CX metric. |

## Products

| Product | What |
|---------|------|
| **AI Agent** | Autonomous ticket resolution |
| **Shopping Assistant** | Proactive pre-sale chat (Convert) — helps customers find products, drives conversion |
| **Macros** | Response templates + bundled actions |
| **Rules / Automation** | Event-triggered workflows |
| **Order Management (OM)** | Native order lookup + actions from within Gorgias (cancel, refund, etc.) |
| **Reporting** | Analytics dashboard (ticket volume, response time, CSAT, automation rate, etc.) |

## Key Metrics Definitions

| Metric | Formula | Why It Matters |
|--------|---------|----------------|
| **Automation Rate** | AI-resolved tickets / total tickets | Primary KPI for AI Agent performance |
| **Coverage Rate** | Tickets AI attempts / total tickets | Shows if AI is being triggered enough |
| **Success Rate** | Tickets AI resolves / tickets AI attempts | Shows AI quality when it does engage |
| **TPO** | Total tickets / total orders | Measures CX efficiency vs business volume |
| **WISMO Rate** | WISMO tickets / total tickets | Indicates tracking/fulfillment communication gap |
| **FRT** | First Reply Time | Speed of first response |
| **Resolution Time** | Time from open to close | Overall efficiency |
