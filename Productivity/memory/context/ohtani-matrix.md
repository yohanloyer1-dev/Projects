## What It Is

A Mandala-based decision governance system (9x9 = 81 boxes). Named after Shohei Ohtani's use of the Harada Method (本田圭佑 Mandala). Used to govern all of Yohan's life, projects, and resource allocation decisions.

---

## Locked Goal

**€50K net/year within 3 years, without full-time employment, working ≤20 hours/week by choice.**

---

## 3 Levers (in order)

1. **Active Cash** → The Gorgias Agency (generates immediate income)
2. **Scalable Assets** → Content (LinkedIn, eventually courses/tools — compounds over time)
3. **Capital** → Investment layer (later, once levers 1 + 2 are running)

## 8 Pillars

*(Full detail to be completed in dedicated ChatGPT session)*

The 8 pillars govern every project, task, and decision. Each item in life maps to one of these pillars.

---

## Decision Filter (apply to every task/project)

1. Which of the 8 pillars does this belong to?
2. Is this Core (must do) or Sandbox (nice to have)?
3. Does this create Leverage (assets, systems, compounding) or just Activity (busy work)?
4. Does it conflict with any constraints (time, energy, focus, values)?

---

## Protect Yohan From

- Over-diversification (too many projects at once)
- Shiny object syndrome (new ideas that dilute focus)
- Parallel execution (doing too many "good" things simultaneously)
- Building activity instead of assets
- Optimizing tactics without linking to pillars

---

## Rules for Claude

- Never suggest multiple new initiatives without filtering through the Matrix first
- Never encourage parallel execution
- Always ask: does this compound, or does it just consume?
- When in doubt, default to: does this serve Lever 1 (Agency) right now?
